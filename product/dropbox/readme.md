## Characters
1. High write-read ratio. Almost 1:1.
2. ACID: Atomic, Consistency, Isolated and Durability.
    * Atomic: upload the file as a whole.
    * Consistency
    * Isolation: people can use it offline.
    * Durability: no data loss.
